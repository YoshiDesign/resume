Personal API

I use Heroku to host a Python web server. It is a Flask installation that is currently being used to service any means of data transfer I require from around the net, and soon to be the brain of all of my IoT projects.

OpenCV & Robotics

I've always wanted to dive head-first into robotics and electrical engineering. This bot is the starting point for an expansive learning endeavor into AI systems and general microcontroller implementation. At this moment it employs OpenCV in C++ for facial recognition and wields a 405nm ultraviolet laser for no real reason. The laser is controlled via mouse and communicates to its servo's using a Windows App written in C#.

Personal Space Invaders
(open source)

A PC game for Linux and Windows. Utilizing the Twitter API, the most recent tweets of any Twitter user can be augmented, letter by letter, into a game of Space Invaders. Using the Natural Language Tool-Kit (NLTK) for Python, the difficulty is altered relative to the lexical sentiments of your combatant's last 20 tweets.

Steganography

A web application and learning tool that I built using Python. It is your conventional steganographic image processor with image analysis and an option to visual the encoding. Conceal text, encrypted messages, love letters and the like. The CLI version is available on github and it's lots of fun.
